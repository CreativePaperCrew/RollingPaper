# 🙏 팀 문화

1. 매일 팀 미팅을 가져요

- 오전 11시에 디스코드에 모여 미팅으로 하루를 시작해요
- 오늘 업무와 업무 진행 사항을 이야기 해요

2. 매주 함께 회고해요

- 매주 일요일까지 주간 회고를 작성해요
- KPT를 작성하여 회고를 진행해요

3. 데일리 스크럼을 작성해요

- 요일 별 각자 할 일과 연락이 안되는 시간을 작성해요

4. 의견을 자유롭게 이야기하고, 피드백을 적극 반영해요

- 추가 구현 사항이나 코드 개선 사항을 이야기해요
